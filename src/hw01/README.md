## 函數重載(Function Overloading)：以參數的(奇/偶)數，使用(*/#)繪製直角三角形圖。

- 請參閱主程式的函數使用方式，設計夠用的『重載』函數 drawLine(...)。
- 本題的解答方式『不可』使用內定參數(Default Parameters)。
